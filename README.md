# Loan_Default_Risk_Analysis using Excel

![Dashboard](https://github.com/Anurag7215/Loan_Default_Risk_Dashboard/blob/main/Dashboard.png)

## Overview
This project focuses on analyzing Loan Default Risk using summary-level financial data.
The dataset contains aggregated metrics such as:

**Average Credit Score

Average Loan Amount

Average DTI Ratio

Average Interest Rate

Distinct Loan Count

The goal of this project is to understand borrower behavior, identify credit risk patterns, and generate meaningful insights that support financial decision-making.
